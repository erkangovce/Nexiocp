# NexioCP

Modern Linux Hosting Control Panel for Ubuntu servers with Turkish language support.

![Dashboard](screenshots/Untitled-1.png)

## Features

- Domain Management
- DNS Management
- Email Management
- FTP Management
- SSL Management
- File Manager
- Backup System
- System Monitoring
- License Management
