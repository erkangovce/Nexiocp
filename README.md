# NexioCP

Modern Linux Hosting Control Panel.

## Features

- Domain Management
- DNS Management
- Email Management
- FTP Management
- SSL Management
- File Manager
- Backup System
- System Monitoring
- License Management

## Installation

wget https://cdn.nexiocp.com/install.sh
chmod +x install.sh
sudo bash install.sh
